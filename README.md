# Chord Piano Position Query

This is a purely front-end chord piano position query tool that can help music learners and piano enthusiasts quickly view the positions of different chords on the piano keyboard.

Supported chord types

Triad
- major triad
- minor triad
- Diminished triad (dim)
-Augmented triad (aug)

Seventh chord
- Major seventh Chord (maj7)
- Minor seventh chord (min7)
Dominant seventh chord (7)
- Half Diminished seventh chord (min7b5)
- Diminished seventh chord (dim7)

Ninth chord
- Major Ninth Chord (maj9
- min9 chord
- Ninth chord (9)

Other chords
- Hang the second chord (sus2)
- Hang a fourth chord (sus4)
- add9 chord
- Major sixth Chord (6)
- min6 chord

How to use

1. Select the root note (such as C, D, E, etc.
2. Select the chord type (such as major, min7, etc.
3. All the notes of this chord will be highlighted on the piano keyboard
